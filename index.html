UPDuino v1.0
A low cost, minimalistic Arduino-like breakout board with Lattice Ultraplus FPGA.

Gnarly Grey UPDuino Board Features

  • Lattice UltraPlus FPGA
  • 5.3K LUTs, 1Mb SPRAM, 120Kb DPRAM, 8 Multipliers
  • Stacks onto Arduino Nano and Arduino Pro Mini Boards (same header spacing)
  • 34 GPIO on 0.1” headers
  • SPI Flash, RGB LED, 3.3V and 1.2V Regulators
  • Open source schematic and layout using free Eagle PCB design tools

UPDuino v2.0
A low cost, minimalistic Arduino-like breakout board with Lattice Ultraplus FPGA and a USB programming device

Gnarly Grey UPDuino v2.0 Board Features:

  • Lattice UltraPlus FPGA
  • 5.3K LUTs, 1Mb SPRAM, 120Kb DPRAM, 8 Multipliers
  • FTDI FT232H USB to SPI Device for FPGA programming
  • 12Mhz Crystal Oscillator Clock Source
  • 34 GPIO on 0.1” headers
  • SPI Flash, RGB LED, 3.3V and 1.2V Regulators
  • Open source PCB design in Eagle and Programming Instructions

UPDuino OV7670 Camera Adapter
Connects the Omnivision OV7670 camera module to the UPDuino FPGA boards. A camera IF to SPI slave with image frame buffer design is provided as well as image extraction instructions for UPDuino v2.0

UPDuino OV7670 Camera Adapter Board Features

  • Easily connects the OV7670 camera module to UPDuino v1.0 or v2.0
  • Provides crystal oscillator footprint for additional low power clock source
